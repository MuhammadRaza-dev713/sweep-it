# Sweep-It — Marketing Site (Next.js + Tailwind)

This repository contains a simple responsive marketing frontend built with Next.js (App Router) and Tailwind CSS.

Getting started (Windows PowerShell):

1. Install dependencies

```powershell
cd d:\sweep-it
npm install
```

2. Run the development server

```powershell
npm run dev
```

3. Open http://localhost:3000 in your browser.

Notes:
- This scaffold uses the `app/` router in Next.js and Tailwind for styling.
- Replace `public/phone-mockup.png` with your own app mockup image.
- To build for production: `npm run build` then `npm start`.
